Recipe Parser

Technologies Used: Python

Utilized natural language processing to parse through recipes on www.recipes.com to get important information. Organized all the ingredients into JSON objects and gave properties like sweetness, sourness, spiciness, bitterness. Then came up with an algorithm to swap the ingredients that are similar in these properties and mix ingredients to make a new dish. The three options are to make the dish healthier, change the cuisine style, and also alter cooking methods.

Features: Web scraper using Beautiful Soup, functions to swap cooking methods and ingredients
